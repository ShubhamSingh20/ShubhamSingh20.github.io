project Golang web automation

Introduction

One of my freelance projects was at a coaching institute which needed to automate the process of sending daily timetables to their students. Up until this point, someone had to manually forward the daily schedule to students sometimes in a group and sometimes even individually to up to 150-200 students. Even students from the same class can have different classes each day due to remedial classes, reassessment examinations, etc.

The crux of the project was to make a program that could take different excel files containing the timetable and contact info of the students as input and render and sent messages to the WhatsApp account of the students.

Main features of the application –

  • Should work with .xlsx, .odt and .csv file extensions
  • Timetable and student can have 1:1 or 1:N relation
  • Should have Whatsapp Login and QrCode creator (similar to WhatsApp web)
  • Program should be in a form of windows executable
  • Ability to parse and extract timetable details such as subjects, timings, and sessions from different files

Technical

  • Go programming language, since the program needs to compile to native code
  • A custom-built WhatsApp “Authentication” and “Send message” service, since WhatsApp didn’t provide an API for developers
  • A separate module for generating Qr code from Whatsapp Web and displaying the Qr Code to a user for scanning
  • After scanning Qr code a session will be generated by Whatsapp which I can use for making subsequent requests for “Send message” service
  • A parser module specifically built as per the structure of the messages and data provided by the client.
