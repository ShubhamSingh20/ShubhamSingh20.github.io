Introduction

One of my freelance projects, DGTOnline is a platform for a Mumbai based competitive exam coaching center. The main purpose of this platform was to provide a simple and easy to use interface that could help the faculty at the institution to easily conduct test with more accuracy and more frequency. At the same time the platform would help students to get accustomed to web based testing portals for these competitive examinations.

Students can login using their respective Google ID and faculties will be given a separate interface for maintaining a list of all the valid students as well as uploading and managing test questions.

Main Features of DGTOnline –

  • Latest question & answer format as per the JEE advance guidelines (numerical and multiple correct MCQs)
  • Save questions for later review
  • Takes backup of all the student answer in case of loss of connection
  • Allow Test retake only for a selected students
  • Using google Auth, thereby more convenient registration process
  • Filtering questions based on subject

Tech stack

  • Python and Django – It’s the core web framework in which the entire backend is written in.
  • DjangoRESTFramework – Certain endpoints are also to be utilized by REST services in later revision of the product
  • Vue JS – Used for implementing dynamic elements in the page like sorting questions, marking questions for later review and timeout timer.
  • Ajax – It’s used in pair with Vue, for making request to django endpoints to populate the Vue data fields.
  • MySQL – Main database used in production

Deployment strategy

  • Digital Ocean – Ubuntu based droplet instance for deploying application
  • Nginx – Used as a proxy server to serve static content and configuring SSL
  • Gunicorn – Used for creating and managing workers
  • EEF Certbot – For generating and renewing HTTPS certificate
  • A custom bash script for automating all the maintenance processes and handling services for any changes
  • Another bash script for creating multiple log files for every change made to application
